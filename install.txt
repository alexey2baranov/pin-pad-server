1. подключите пин-пад через usb-кабель
2. Установите из папки dist драйвера пин-пада
3. установите из папки dist программу arcus в текущую папку поверх имеющейся
4. установите node.js https://nodejs.org/en/ последнюю стабильную версию
5. добавьте pin-pad.bat в автозагрузку и выберите в качестве рабочей папкои текущую папку
